## 数据标注格式

### 分类
- 目录文件结构：
```
dataset
├── ImageSets
│   ├── train.txt
│   ├── val.txt
└── JPEGImages(*.jpg,*.png,*.bmp,*.jpeg)
```

- 标注格式：无

### 检测
- 目录文件结构：
```
dataset
├── ImageSets
│   ├── train.txt
│   ├── val.txt
├── JPEGImages(*.jpg,*.png,*.bmp,*.jpeg)
└── Annotations(*.json)
```
